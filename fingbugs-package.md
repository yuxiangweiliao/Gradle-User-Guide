# FindBugs 插件  
  
FindBugs 插件使用 FindBugs 对项目的 Java 源文件执行质量检查，并从检查结果中生成报告。

## 用法  

要使用 FindBugs 插件，请在构建脚本中包含以下语句：

**示例 使用 FindBugs 插件**

build.gradle  
  
```
apply plugin: 'findbugs'  
```  

该插件向你的项目添加了大量的执行质量检查的任务。你可以通过运行 gradle check 执行检查。
